# Privacy Policy — DRS Pro

DRS Pro is designed for **local-only** operation.

## Data collected

- **None by default.** DRS Pro does not send analytics or telemetry to external servers.

## Data stored on your computer

| Location | Contents |
|----------|----------|
| `%LOCALAPPDATA%\DRS\settings.yaml` | Your preferences |
| `%LOCALAPPDATA%\DRS\calibration\` | Stump line calibration per ground |
| `%LOCALAPPDATA%\DRS\clips\` | Video clips you save |
| `%LOCALAPPDATA%\DRS\sessions\` | Session JSON logs |
| `%LOCALAPPDATA%\DRS\logs\drs.log` | Application log file |

You may delete this folder at any time to remove all local data.

## Live camera mode

If you use a USB webcam, video stays on your machine unless you explicitly save a clip.

## Contact

For privacy questions, contact your software vendor.
